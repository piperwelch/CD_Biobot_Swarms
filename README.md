# AI Swarm Design 

This repo contains code and in vitro data for AI design of Anthrobot swarms. 

## Requirements 

Python 3 
CUDA 10 

## Getting started 

To begin evolution, run the following commands:

``cd swarms/in_silico``

``sbatch afpo_begin.sh ``
